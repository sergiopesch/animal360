# animal360

A Salesforce application for animal welfare organisations.

## Overview

`animal360` is a Salesforce-based solution designed to help animal welfare organisations manage cases, track animals, and improve care coordination.

## Features

- Centralised animal records
- Case management support
- Organisation-focused workflows
- Salesforce-native integration

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/sergiopesch/animal360.git
   ```
2. Open the project in your preferred Salesforce development environment.
3. Review the source and deploy as needed.

## Usage

This project is intended to be installed and configured within a Salesforce org. Follow your normal Salesforce deployment process for metadata and Apex components.
